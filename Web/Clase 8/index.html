Ejemplos de ecuaciones en HTML

Para escribir ecuaciones en una página web en html se puede usar MathJax. La fuerza eléctrica se puede calcular como $$\vec{F}=q\vec{E}$$

La fuerza eléctrica entre dos cargas es: $$\vec{F_{12}}=\frac{1}{4\pi\epsilon_0}$$

La circuitazione del campo elettrico é:

$$\Gamma(\vec{E})=\sum_{i=1}^n\vec{\Delta l}_i\cdot\vec{E}_i$$

La solución de una ecuación cuadrática de la forma \(ax^2+bx+c=0\) es la siguiente: $$x_{1,2}=\frac{-b\pm\sqrt[3]{b^2-4ac}}{2a}$$
